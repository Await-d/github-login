# 技术栈

## 后端技术
- **Node.js**: JavaScript运行时
- **TypeScript**: 类型安全的JavaScript
- **Express**: Web应用框架
- **SQLite3**: 轻量级数据库
- **bcrypt**: 密码哈希加密
- **express-session**: 会话管理

## TOTP相关
- **speakeasy**: TOTP/HOTP生成和验证
- **qrcode**: 二维码生成

## 前端技术
- **原生HTML/CSS/JavaScript**: 简单的Web界面
- **CORS**: 跨域资源共享

## 开发工具
- **ts-node**: TypeScript直接运行
- **Jest**: 测试框架
- **@types/***: TypeScript类型定义

## 项目结构
```
src/
  ├── models/     # 数据模型
  ├── routes/     # 路由处理
  ├── middleware/ # 中间件
  ├── utils/      # 工具函数
  └── types/      # 类型定义
public/
  ├── css/        # 样式文件
  └── js/         # 前端脚本
```
